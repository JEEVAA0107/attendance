# SmartAttend Hub

Modern attendance management system for colleges with automated tracking, real-time analytics, and seamless data export.

## Features

- **Smart Attendance Tracking** - Automated attendance management
- **Real-time Analytics** - Live dashboard with attendance insights
- **Student Management** - Comprehensive student database
- **Data Export** - Export attendance data to Excel
- **Responsive Design** - Works on all devices

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd smart-attend-hub

# Install dependencies
npm install

# Start development server
npm run dev
```

## Technologies Used

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
- **Charts**: Recharts
- **Forms**: React Hook Form with Zod validation

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
